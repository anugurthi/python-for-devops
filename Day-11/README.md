# Day 11 Â· Automating GitHub with Python

Today we will learn how to create GitHub repositories using Python instead of clicking through the web interface.

## What You Will Learn

-   **GitHub API:** How to create a GitHub repository programmatically.
-   **API Tokens:** How to authenticate with GitHub using a personal access token.

## Instructions

1.  **Setup:** Create a GitHub personal access token at https://github.com/settings/tokens
2.  **Save Token:** Store it as an environment variable:
    ```bash
    export GITHUB_TOKEN="your_token_here"
    ```
3.  **Run Example:** Look at the example scripts in `examples/` to see how to create repos.

-   [ ] I have a GitHub personal access token.
-   [ ] I understand how to use the token to authenticate API requests.
